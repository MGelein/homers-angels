Guides

A few documents to help you learn or remember the steps of the editorial process.

N.B: These documents are written in Dutch. Apologies to our foreign colleagues. An English guide MAY be written or not... Time is hard to come by.

  • Installation - The installation process of the tools that are required for collaboration in the HMT project.
  • Encoding - An essential part in the installation procedure for all the contributors using Dutch Windows versions.
  • Git - A basic description of the usage of Git commands for version control.
  • Workflow - An outline for the basic workflow we use during editing.
  • Paleographic Guide for Venetus A - How to read the various squigly lines that are said to correspond to the characters in the Greek alphabet.
