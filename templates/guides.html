Guides

To aid newcomers to the project as well as seasoned veterans who have been through manuscript hell and back we've compiled a few guiding documents to help you on your way to learn or remember the steps of the editorial proces for the Homer Multitext Project.

N.B: These documents are written in Dutch. Apologies to our foreign colleagues. An English guide MAY be written or not... Time is hard to come by.

  • Installation - Describes the installation process of the tools that are required for collaboration in the HMT project.
  • Encoding - This is an essential part in the installation procedure for all the contributors using Dutch Windows versions.
  • Git - This document contains a basic description of the usage of Git commands for version control.
  • Workflow - In this document you can find an outline for the basic workflow we use during editing.
  • Validatie - Reading this document will teach you how to use the SBT and Scala to validate the documents on your own pc.
  • Paleographic Guide for Venetus A - How to read the various squigly lines that are said to correspond to the characters in the Greek alphabet.
