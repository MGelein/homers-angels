Tools

During the editorial process you sometimes have to do some tasks that most ordinary people find boring. While attending the CHS HMT summer seminar of 2017 the tools that are listed below were made by Mees Gelein to automate some of the more menial tasks (who likes counting letters for example?) and to speed up the editing process.

Suggestions

If you have any suggestions for new tools that you think might benefit the HMT project or if you have feedback concerning the usage of the current tools, do not hesitate to contact us. You can find our contact details on our Contact page.

Please remember that any and all tools made for this project are made in the little spare time a programmer can spare. Bugs will be present, probably alot. Be prepared, save often!

URN Convertor

The URN Convertor tool provides you with a quick lookup tool to search through IMG urn's, place URN's and personal URN's.

Continue to the Tool
Paleo Corrector

The Paleo Corrector tool provides you with a quick check, correct and counting tool to check, correct and count in your .cex files for the HMT project.

Continue to the Tool
Image Request(or)

The Image Request utility lets you load regions created by the Image Citation Tool and save them or link to them.
